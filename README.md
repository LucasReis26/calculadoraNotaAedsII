# Calculadora de Nota de AEDs-II

Programa em C com o intuito de calcular a sua nota atual de AEDs-II.

## Como utilizar

1. Clone o repositório na sua máquina e acesse ele via terminal.

1. O repositório possui o arquivo main.c e o programa compilado chamado exec, caso você utilize uma distro linux você pode apenas executar o programa digitando ./exec dentro da pasta do repositório.

1. Caso sua máquina seja windows você deve compilar novamente o main.c e executá-lo.

> Notas dos Quizzes
>
> Esses pontos estão imbutidos na nota de quizzes das notas, essas notas estão separadas no próprio canvas como Prova X - Quiz, onde X é a prova que você irá inserir.
